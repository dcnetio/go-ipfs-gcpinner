# go-ipfs-pinner

## Background

The gcpinner adds GC function on the basis of  [go-ipfs-pinner](https://github.com/ipfs/go-ipfs-pinner)  

## Install

Via `go get`:

```sh
$go get github.com/dcnetio/go-ipfs-gcpinner
```

> Requires Go 1.13
